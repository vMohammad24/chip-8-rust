# A simple chip-8 emulator
I wanted to learn rust so this seemed like a fun starting project.

### Currently implemented
- Chip-8
- Super-Chip


### Instructions
It is mostly meant for myself, but if you build it you can point it to a rom.
Like this: `chip-8 roms/tetris.ch8`
